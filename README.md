Application Desktop - AV Admin Config

Sobre o projeto
Aplicação desktop desenvolvida em Java utilizando JavaFX e FXML, estruturada no padrão MVC.  

O sistema possui navegação entre múltiplas telas e organização por controllers, permitindo cadastro e testes de integração com banco de dados.

Projeto desenvolvido com foco em orientação a objetos e estruturação de aplicações desktop.

---

## 🚀 Tecnologias Utilizadas
- Java
- JavaFX
- FXML
- Padrão MVC
- Integração com banco de dados
- Git

---

Estrutura do Projeto

Organização baseada em separação de responsabilidades:

Exemplo:
- MenuController
- Telas FXML
- Navegação entre cenas utilizando FXMLLoader

---

Funcionalidades

- Navegação entre telas
- Menu principal
- Cadastro de dados
- Integração/teste com banco de dados
- Encerramento da aplicação
